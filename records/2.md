## Step 2: /workflow-propose

**Command:** `opencode run "/workflow-propose user-feedback-form"`

**Expected:** Convert draft to OpenSpec documents

**Actual Result (First Attempt):**
```
✗ Failed to create proposal: Command failed: openspec propose "user-feedback-form"
Error: unknown command "propose" for openspec
```

**Root Cause Found:**
```typescript
// workflow-tools.ts line 314
const { stdout } = await execAsync(`openspec propose "${draftName}"`, {
```
**Bug:** Should be `openspec new change` not `openspec propose`

**Fix Applied:**
```typescript
// Changed from:
execAsync(`openspec propose "${draftName}"`)
// To:
execAsync(`openspec new change "${draftName}"`)
```

**Second Attempt Result (after fix):**
- Successfully created `openspec/changes/user-feedback-form/`
- Created 4 artifacts: proposal.md, design.md, specs/feedback-form/spec.md, tasks.md

## Step 5: /workflow-start

**Command:** `opencode run "/workflow-start"`

**Expected:** Linus agent executes tasks in dependency order

**First Attempt Result:**
```
Linus orchestration output displayed correctly
But subagent exceeded 200 tool calls - infinite loop detected
Task was cancelled
```

**Root Cause Analysis:**
1. The `workflow-start` tool itself works - it outputs task info
2. The issue was in how Linus agent was invoked via `session.prompt()`
3. Linus agent prompt was not being passed correctly

**Issue #1:** Agent prompt not configured in op-test project
**Issue #2:** OpenCode uses `agent` key (singular), oh-my-openagent uses `agents` key (plural)

**Discovery:**
```
oh-my-openagent (custom plugin): oh-my-opencode.json with "agents": {...}
OpenCode (standard): opencode.json with "agent": {...}
```

**Fix Applied:**

1. Created `.opencode/agents/linus.md`:
```markdown
---
description: Linus - Workflow orchestrator that executes tasks from bd queue
mode: subagent
---

You are Linus, the workflow orchestrator...
```

2. Updated `.opencode/opencode.json`:
```json
{
  "agent": {
    "linus": {
      "description": "Linus - Workflow orchestrator that executes tasks from bd queue",
      "mode": "subagent",
      "prompt": "{file:.opencode/agents/linus.md}"
    }
  }
}
```

**Second Attempt:** 
```
opencode run "/workflow-start" --agent linus
```

**Result:** Linus agent successfully loaded and began task execution

## Key Findings

### Bug #1: `openspec propose` → `openspec new change`

**File:** `packages/plugin-core/src/workflow-tools.ts`  
**Line:** 314  
**Severity:** Blocking

```typescript
// WRONG:
await execAsync(`openspec propose "${draftName}"`)
// RIGHT:
await execAsync(`openspec new change "${draftName}"`)
```

### Bug #2: Agent Configuration Key Mismatch

**Issue:** oh-my-openagent uses `agents` key, OpenCode uses `agent` key (singular)

**Discovery from research:**
| Project | Config File | Key |
|---------|------------|-----|
| oh-my-openagent | `oh-my-opencode.json` | `"agents": {...}` |
| OpenCode standard | `opencode.json` | `"agent": {...}` |

### Agent Configuration Schema (OpenCode Standard)

```json
{
  "agent": {
    "linus": {
      "description": "Required - describes what the agent does",
      "mode": "subagent",  // "primary", "subagent", or "all"
      "prompt": "{file:./agents/linus.md}",
      "tools": {
        "write": true,
        "edit": true,
        "bash": true
      },
      "permission": {
        "task": {
          "coder": "allow"
        }
      }
    }
  }
}
```
